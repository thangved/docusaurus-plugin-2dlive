> A live2d docusaurus plugin

## Installation

```sh
npm i docusaurus-plugin-2dlive
```

or...

```sh
yarn add docusaurus-plugin-2dlive
```

## Usage

In file `docusaurus.config.js`

```js
...
module.export = {
    ...
    plugins: [
        ...
        'docusaurus-plugin-2dlive',
        ...
    ],
    ...
}
...
```
